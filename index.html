Badge for Website Design

Ethan Kahn

Description

For my Music Production Badge, I began developing a website to present my work. However, once I heard how many presentations the judges had to go through, I decided to spice mine up and add some interesting features. What came of it was an elaborate website filled with fun, interactable ways of presenting information.

Steps

For each page, I followed specific steps to get to the page I wanted

  1. I start with choosing an interesting idea for a webpage. This usually comes just from me, but occasionally I look things up for inspiration.
  2. I then begin creating a shell of a page using HTML. A minimal amount of written content goes into this part: that comes later.
  3. Next, I start stylizing using CSS.
  4. After that, I add in whatever functionality via JavaScript.
  5. Finally, I add the actual written content to the page.

Reflection

In complete honest, I've had a snobby look at web developement for a long time. It's definitely an artsy part of the coding world, so my algorithmic centric mind would look down at web developers. However, this experience has opened my eyes to how difficult it can be to create a website. I hope to coninue developing websites in the future.
